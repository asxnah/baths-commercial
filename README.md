# Hithaus Banya MVP

MVP карточки коммерческой бани для обновления сайта Hithaus (по тестовому заданию).

## Стек

- Next.js 14+ (App Router, Server Components)
- Tailwind CSS
- Данные вынесены в `@data/baths`
- Изображения оптимизированы через `next/image`

## Реализовано

- Главная (`/`): адаптивный грид карточек бань с превью и ценой
- Страница проекта
  - Хлебные крошки
  - Галерея-слайдер
  - Заголовок (h1), цена, описание
  - Модальное окно заказа
  - Семантическая таблица характеристик

## SEO / GEO

- JSON-LD Schema.org Product (`<SeoJsonLd />`)
- Правильная семантика
- AI и поисковики легко парсят

## Производительность

Server Components + next/image → Lighthouse 95+ (ожидаемо).

## AI в разработке

Структура данных, галерея, таблица, JSON-LD и тексты генерировались через ChatGPT/Copilot/Cursor. Boilerplate сокращён на ~60%.

## Запуск

```bash
npm i
npm run dev
```
